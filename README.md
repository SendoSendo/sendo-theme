# Sendo theme

Advanced UI/UX theme for Sendo (accessibility, motion, design tokens). Uses the Sendo logo (flying box + SENDO wordmark).

## Logo

Upload your Sendo logo (flying box + SENDO wordmark) in **Theme settings → Logo & favicon → Logo**. Until then, the store name is shown. Optional: add `sendo-logo.png` to `assets/` and you can wire it as a default in the header.

Brand colors (neon green, orange, dark) can be set in **Theme settings → Colors**.
